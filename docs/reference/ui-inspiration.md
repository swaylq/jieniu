# UI 风格灵感 · 好看的设计总结

> 自驱循环产出（loop `cmptloop`）。目的：收集好看的财经/工具类 App UI，提炼可落地到 **解牛** 的设计。
> **解牛现状**：琥珀 `#f5a623` 主色 + 灰底卡片 + 红涨绿跌 + class 暗色 + Tailwind v4 语义 token + 编辑型财经风（详见 `ui-style.md`）。
> 竞品的功能优点见 `competitor-research.md`。

## 提炼维度

- 配色 / 排版 / 信息密度
- 资讯卡 / 信息流设计
- 个股·实体页布局
- 数据可视化（走势图 / 迷你图 / 热力 / 榜单）
- 重要度·标签·状态的视觉编码
- 微交互 / 空态 / 暗色 / 品牌质感

## 富途牛牛 / moomoo（UI 标杆，与我们编辑型财经风最契合）

- **四条资讯设计原则**：沉浸浏览 / 轻量精致 / 清晰流畅 / 高效获取；铁律 **"一页只说一件事、一屏不超过 3 个内容"**。
- **卡片：扁平、去边框、去分割线、留白分区**；Banner 用贴合视口的**圆角大卡**，靠图片+留白建层级；**圆角是全局视觉载体**。
- 排版为阅读优化：正文加大字号/行距/段距，中文 **两端对齐(justify)**；专栏加粗标题区分摘要。
- 快讯 **时间轴**：时间戳节点用**高亮色**做锚点兼分割线；次要入口弱化。
- 标签分级：重要标签图形化，次要纯文字。
- 深色主题**专门调校**缓解盯盘疲劳；按"功能/业务属性"组织"颜色包"，桌面三层色彩系统。
- "高级感"来源：**极致克制的密度 + 圆角/留白/去线的扁平体系 + 为长阅读调校的字排配色**，非炫技动效。
- → 解牛落地：圆角大卡 + 留白 + 去线（`rounded-2xl`+`gap`/`py`、删 `border`/`divide`）；阅读页 `text-base leading-relaxed`+段距+中文 `text-justify`；快讯时间轴。

## 财联社（重要性视觉编码 + 电报流）

- **电报流**：极高信息密度时间流，一屏多条；每条=时间戳（如 `17:43`）+短句，头条带标题+摘要；按日期分段可回看。
- 配色克制专业（web 实测）：重要用 **橙红 `#ef9524`（加红）**、背景浅灰 `#f9fafc`、正文深灰 `#222` —— 低饱和阅读友好，**红色只用于"重要性信号"不满屏渲染**。
- **重要性可视化分层**：加红文字 + 涨停红标/概念灰标 + 排名升降箭头。
- 二级 tab 过滤（全部/加红/公司/看盘/…）。
- 多模态：文章/电报/直播支持**语音播报** + "语音电台"悬浮条；视频小窗后台播放。
- 分享：长文一键生成**精美长图海报**（配图+标题摘要+二维码）。
- 顶部/侧栏常驻四大指数行情条（只读）+ 市场热度计。
- → 解牛落地：**用琥珀 `#f5a623` 作"加红"单一强调色**标记重磅；时间流密度控制；指数条（已做）；语音/海报分享（可选）。

## 雪球（内容社区式布局）

- **极简白底 + "平静的蓝色调"**，每屏只突出最重要内容 —— 像内容社区不像交易终端（与同花顺/东财满屏红橙反差）。
- 涨跌 **红涨绿跌且可设置切换**（照顾港美股用户）。
- 信息组织：**纵向滚动分区（一屏一主题卡片）+ 横向 tab/滑动**，信息极多但靠折叠分层不乱。
- Feed 卡片：头像/名→正文摘要→**关联股票标签（点击直达个股页）**→操作条；图文/长文/视频统一进流。
- 走势图交互：**长按查看某时刻股价**（十字游标），横屏看大图。
- 夜间模式（可关"跟随系统"）。
- → 解牛落地：白底克制卡片化"阅读产品"观感；实体页"纵向分区 + 横向 tab"手势。

## 同花顺（反面参照：密度过高）

- 配色克制内容优先：底部 tab 浅灰底+深灰字 + **橙色选中态**，弱化导航；橙红主色（涨红跌绿）。
- AI 输出（问财）：PC "左会话导航 + 右内容区"多线程；结果 **表格为主** + 关键数据自动出图（K线/柱状）+ 重点高亮，主打"降低信息处理负担"。
- 数据可视化：资金流用 **地图面积** 表流入流出、红绿深浅表涨跌。
- **反面教材**：信息密度过高、模块视觉区分低、层级深、新手易迷失 → 解牛反向做 **轻 + 聚焦 + AI 优先**。

## 东方财富 / 妙想 AI

- 妙想首页：浅色系、**只留一个核心搜索框 + 快捷提问标签**（"行情解读/板块分析"）—— 降门槛、功能高度聚合。
- **对话流气泡式**：问答分屏；重点数据**自动出可视化图表**（K线/柱状）+ 关键信息高亮；**"追问"按钮**一键延展。
- 主 App：清新明快 + 大面积白底、磨砂玻璃图标 + 弥散投影 + 留白（"轻松无压"）。
- 深色模式**可定时开启/关闭**，涨红跌绿保持对比。
- → 解牛落地：解读做对话流气泡 + 关键信息高亮 + "追问/换个视角"按钮；首页快捷标签聚合。

## 华尔街见闻 & 格隆汇/智通（编辑型专业媒体）

- **快讯 vs 深度视觉切割清楚**（华尔街见闻）：快讯=纯文本+时间戳极简时间流；文章=缩略图+作者署名+时间卡片；"大家都在搜"用大图轮播做编辑精选、与时间流区隔。图片驱动 + **统一缩略图尺寸**利于扫读。
- 克制专业、去娱乐化，分栏布局，信息密度高但不杂。
- 桌面 **widget**（当日 Top 快讯/主题进展）—— 高频轻量心智，验证对留存有效。
- 深色双模式；**戒鉴：华尔街见闻深色黑白对比偏高刺眼 → 解牛暗色压低纯黑/纯白明度**。
- 格隆/智通：7×24 时间戳电报流 + 红绿涨跌 + 重要加粗标红分级；格隆长文偏"研究报告"排版、智通更"新闻客户端"频道 Tab。共同短板=入口多、层级深、新手被淹 → 解牛做"轻/聚焦"减法。

## 国际：Robinhood · Koyfin · Seeking Alpha（视觉标杆）

- **单一强调色 + 语义色分工**：Robinhood 一个绿；SA/Koyfin 一条"红→琥珀→绿"语义尺贯穿评级 pill/gauge/字母徽章。我们 amber 主色+up/down 语义色路子一致。
- **⚠️ 颜色坑**：SA/Robinhood 绿=好/涨；我们红涨绿跌。**红/绿只给真实涨跌；评分/分位/记分卡用 amber 深浅或中性灰阶**(与买卖划清界限+合规)。
- **同轴并置读分歧**(SA 三 gauge 共尺、Koyfin 联动色组)：把多个判断放同一坐标系比较。
- **记分卡=可扫徽章行 + 一键下钻到事实**(SA 字母徽章、Koyfin 0-100 分位)：顶层极简、深度可展开。
- **渐进披露**(Robinhood 长滚动、Koyfin 用户自控密度)："干净"=用户可控的密度，不是藏数据。
- **走势图去杂**(Robinhood)：无网格细线 + **末端发光点** + **虚线基准线** + 区域填充 + 拖拽 crosshair + 时间药丸——移动端图表黄金范式。
- **暗色=深藏蓝非纯黑**(Koyfin Midnight Blue)更高级；主题宜少。
- **表格纪律**：等宽数字(已有 `.tabular`)、红绿只标涨跌列、**别做单元格渐变**(连 Koyfin 都没做)。
- **动效克制**：Robinhood confetti 被监管打→动效只做"帮助理解"不做"庆祝/怂恿"(契合编辑型定位)。
- **无障碍**：红涨绿跌需配色盲开关(Robinhood Accessible colors)。
