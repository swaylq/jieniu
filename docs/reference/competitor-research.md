## 财联社（Cailianpress）— 一手快讯/电报龙头（"一手"最直接对标）

上海报业主办；同时是 B 端通讯社（毫秒同步 150 家金融机构，反哺 C 端权威背书）。

**核心优点**
- **电报/快讯实时流**（定义品类）：7×24、日均近千条、"快短准"电报体（时间戳+一句话）；自研检索从 3 万+ 信源毫秒采集；电报追时效 / 要闻深度频道追确定性。
- **电报二级过滤流**：全部 / **加红** / 公司 / 看盘 / 港美股 / 基金 / 提醒 ——「加红(重磅)」本身是可独立筛选的流。
- **新闻→个股结构化关联**：每条带 `stock_list` 关联个股；热股榜涨停红标/概念灰标+排名升降；**题材库/话题**聚合专题（带关注量）。
- **盘面直播**：时间流做盘前（题材挖掘/避雷针）→盘中（异动原因一键直达/整点快评/主力资金）→盘后（涨停/连板分析）。
- **重磅加红 + 分级推送**（重要弹窗/普通静默）+ 关键词监控提醒。
- **关注频道双流**：我的关注（订阅）+ 最新动态（热门）。
- **AI**：财跃大模型；**AI 小财神**（AI 热点解读——从行业格局/产业链上下游/股权关系推演一条新闻利好利空）；小财神 Pro（半小时出研报，FinResearchBench 国内第一）；**电报解读**（记者+编辑人工：供应链扰动→涨价逻辑→产业链图谱→点名关联公司）。

**可借鉴点（解牛）**
1. **"重磅"视觉分级 + "只看重磅"独立流 + 分级推送**（重要弹窗/普通静默，可选阈值）——强化"一手"，成本低感知强。我们已有 importance/eventType，可做"只看重磅"筛选流。
2. **新闻实体化标签→点击进聚焦页**（个股/题材/板块，状态轻量色标）——把一手流升级为可导航知识网。我们已有实体标注，可强化题材聚合页 + 关注量。
3. **关注驱动"聚焦"双流**（订阅 题材+个股+**大师视角**；我的关注/最新动态双流+订阅提醒）——把"大师视角"也做成可订阅维度。
4. **"新闻→结构化解读卡"**（背景→影响面→产业链上下游→关联公司利好/利空推演），用大师人设语气替代记者口吻；红线：只做逻辑/影响面/观察点，禁买卖/目标价/涨跌预测。**这是相对财联社 VIP 荐股内容的合规差异化护城河。**
- 附：语音播报/语音电台（通勤）、解读卡一键长图海报（带二维码）分享裂变。

## 雪球（Xueqiu）— 实体聚合页 + 内容社区

极简白底"平静蓝"，像内容社区而非交易终端。

**核心优点**
- **个股页=一站式实体聚合页**：纵向滚动分区（行情→风向标→资金→资讯→公告→资料→财务→研报）+ 横向 tab 切讨论（最新/推荐/精华）。**资讯与观点在同一实体页并置。**
- **自选股=新闻+观点订阅**：关注一只股 = 自动订阅其精华观点流；支持拍照/OCR 批量导入自选。
- 首页三档：**热门(编辑) / 推荐(算法) / 关注(纯订阅)** 左右滑。
- **组合**：用实盘净值/收益/调仓给观点背书（交易属性不可照搬，但"可验证数据背书观点"思路可迁移）；大V 五维能力评分。
- 7×24 快讯 + 话题/热帖 + 热股榜；股价提醒。
- **AI（2025）**：**雪球思考**（站内+全网双模式，整合财报/研报/社区，出竞争格局图，**自动对比"机构 vs 散户"视角**）；**AI 解读**（个股页：快讯摘要/情绪分析/财报解读/趋势判断，底部免责"基于用户讨论生成、不代表官方"）；AI 公告摘要；"AI 帮你问"入口。

**可借鉴点（解牛）**
1. **实体页做信息聚合中枢，但重排为"新闻优先"**（我们是新闻 App：第一屏放一手新闻流，行情只留极简只读条；下接 大师解读→公告(含 AI 摘要)→财务快照）；删盘口/资金/跟投/下单。
2. **AI 大师解读做实体页"一等公民" + 抄免责范式**：(a) 结构化解读卡固定骨架（一句话观点/估值视角/商业模式/风险）同结构可横向对比；(b) **多大师视角并排**（巴菲特 vs 芒格 vs 林奇 看同一新闻的分歧与共识）——比雪球"机构vs散户"更有记忆点；(c) 每条挂"AI 以大师方法论风格生成，仅供参考非投资建议"。
3. **"关注即聚焦"**：自选(个股+行业+概念+关键词)驱动个性化流；**"股价提醒"改造为"事件/新闻提醒"**（重大公告/突发/财报/解读更新/舆情异动触发，不设到价）；首页沿用三档。
4. **"一键读懂"入口**："用大师视角读懂这条""这条对我自选有何影响"，把一手新闻与大师解读在交互上缝合。

## 同花顺（THS / 10jqka）— AI 功能标杆

MAU ~3400 万；All in AI，问财是国内金融第一 AI 对话产品。

**核心优点**
- **问财 i问财**：自然语言选股/解读；**圈选资讯/行情/K线→让 AI 解读对比**（与"大师解读"最直接对标）；2025 升级为自主规划 Agent。付费 ARPU ~1572 元/年（AI 解读可付费）。
- **HithinkGPT**：首家备案金融大模型；自训 1B 小模型做合规审查（5 类 31 项，**风险召回 99.5%**，"内容可控生成"）。
- **AI 盘中异动解读**：每条=异动标签 + **"概念+事件"触发原因** + 受影响个股(直链) + 分钟级时间戳 + 可展开。事件→个股是全 App 信息主线。
- **AI 内容自动生成**：午评/收盘点评/脱水研报/热点专题批量产出。
- 诊股五维（投顾/技术/资金/消息/基本面）；资讯模块化（要闻/滚动/机会/自选）。
- 短板：信息密度过高、层级深、新手易迷失 → 可反向差异化。

**可借鉴点（解牛）**
1. **"概念+事件"AI 归因** → 每条新闻/公告自动打"主题+事件类型"标签直链个股，做"新闻→影响了谁"因果卡片（只事实归因，不预测涨跌）。
2. **"圈选内容→唤起大师解读"招牌交互**（长按/点解读→persona 结构化：结论先行+要点高亮+可展开）。
3. **AI 公告脱水摘要 + 事实倾向标注**（利好/利空/中性+关键数据+不确定性，标"非投资建议"）。
4. 反打其短板：轻/聚焦/AI 优先，首屏"一手时间线 + AI 摘要卡"。
5. **合规过滤层必做**（拦买卖建议/目标价/收益承诺，对标 99.5% 风险召回）。

## 富途牛牛 / moomoo — UI 精致度 + AI 早晚报

**核心优点**
- **资讯是一等公民**：底部导航"资讯"居中主入口（非埋在个股页）；多维切分（要闻/快讯/专栏/专题/日历/自选/异动）+ 7×24 全球快讯。
- **个股页新闻权重 > 财务**：详情 tab 明确"新闻/评论 优先级 > 资金/分析/公告/简况"；财务按业务/产品/地区三维拆；同行业个股自动对标。
- **财经日历做决策锚点**（财报/分红/新股/经济数据/事件/休市 + **倒计时**，事件后自动刷新）。
- 行情首屏零下钻；K 线 6 种线型；自选 12 分组每组自定义色。
- **牛牛AI（接 DeepSeek-R1）**：**AI 异动解读→早报/晚报/周报**（开盘前+收盘后各一版，休市日也更新）；**条件化展示——没料就不显示**（主动降噪不硬凑）；解读可点击下钻；moomoo 版聚合 200+ 信源出一页纸摘要。**合规话术**："AI 不是投顾、不提供直接建议、增强而非替代决策"（现成模板）。

**可借鉴点（解牛）**
1. news feed 套 **扁平卡 + 留白分区 + 一屏≤3 条**（去嵌套/描边/分割线，圆角大卡靠留白+缩略图分区）。
2. 一手快讯用 **时间轴**，时间戳做彩色锚点（节点线兼分区）——契合"一手""正在发生"感。
3. **大师解读=专栏卡范式**：大师头像/角标 + 加粗标题 + AI 摘要。
4. AI 解读做实体页底部 **早报/晚报/周报条件卡——没料不显示**；借其免责话术。
5. 阅读排版：正文加大字号/行距/段距 + 中文两端对齐；标签分级（重要图形化/次要文字）。
6.（可选）财经日历 + 倒计时作为资讯侧决策锚点（纯展示，可按自选过滤→反哺聚焦）。

## 东方财富（East Money）— 快讯结构化 + 股吧 + 妙想 AI（信源溯源）

**核心优点**
- 7×24 快讯 **3 秒刷新** + 语音播报 + 桌面提示；按维度切分（焦点/股市直播/上市公司/地区/全球央行/经济数据/商品外汇债券）—— 结构化信息流。
- **股吧**（护城河社区）：**个股吧**为单元，围绕标的聚合 资讯+研报+公告+可视化财务+讨论；强内容治理（持续清理非法荐股）。
- **妙想 AI**（与我们最相关）：
  - **动态资讯解读**：实时抓新闻/研报/公告 → 摘要 + **情绪分析** + 自动关联（毛利率/竞对/机构评级变化）。
  - **可信度机制**：金融大模型 + **"信源分级—逻辑拆解—多维验证" + 事实溯源通道**（解决 AI 金融解读"敢不敢信"）。
  - **舆情弹窗预警**：分钟级监控 股吧/雪球/社媒 → 异常信号弹窗；**多维标签矩阵 + 智能事件关联链**（零散消息串成事件脉络）；响应 ~0.7 秒。
  - 对话流气泡 + **追问按钮** + 快捷提问标签；全用户免费、股吧内嵌入口无需切 App。
- 数据中心（龙虎榜/资金流/研报/机构调研/财经日历）；自选跨端同步。

**可借鉴点（解牛）**
1. **每条新闻/公告内嵌"一键大师解读"**（对标妙想动态资讯解读）：输出 摘要+情绪标签+关联（相关个股/历史相似事件/机构评级变化），用大师视角套框架；借 **信源分级+事实溯源**（每条标信源可回溯）解决可信度。
2. **"聚焦"= 事件级舆情弹窗**（对标舆情预警+事件关联链）：关注标的/主题有高价值事件→弹窗；合规改造：只报"事件+多方观点+大师解读"，**绝不含买卖**；舆情只做情绪标签不出信号。
3. **对话式解读 UI：气泡流 + 高亮 + "追问 / 换个大师视角再讲一遍"**（巴菲特 vs 林奇 vs 达利欧对同一条新闻）—— 零合规风险的差异化杀手交互。
4. 首页 **快捷标签 + 高度聚合**（少即是多）：默认"今日焦点/我的关注解读/大师今日观点"，AI 解读放首屏一等公民。
5. **显式产品边界 + 内容治理**（"只做信息解读不构成投资建议"做成产品边界，解读卡带免责+信源）—— 合规即信任卖点。

## 格隆汇 & 智通财经 — 内容分层 + 题材/事件锚点（AI 解读空白）

**格隆汇**（全球投资研究平台，港美股起家）
- **内容五层递进**：快讯/电报→事件→深度→主题/题材→专栏，同一件事从快讯一路下钻到深度长文。
- **题材库**：热点做成题材聚合页（题材热度+涨跌表现+关联个股清单）——主题聚合做得最实。
- **专栏+社区**："多方观点并列"（大V/研究员），官方卖点"聆听大V声音、延伸观点维度"。
- 可控推送：**自选内容类型 + 自选频率**；桌面自选小组件。

**智通财经**（港美股原创资讯，编辑部驱动，日均原创 1000+/快讯 5000+）

## 华尔街见闻（Wallstreetcn）— 实时快讯 + 内容分层 + 主题追踪 + AI 四维

（国内首创"实时新闻"模式；我们已用它做源，这里看其产品设计）

**核心优点**
- **实时快讯流**（招牌）：7×24 独立频道，短+精确时间戳+纯文本倒序；**桌面 widget**；推送"快又精准"。
- **主题订阅实时推送**：订阅 热点行业/细分资产/市场核心风险点，命中即推（精准低噪，非全量）。
- **内容分层**：快讯(秒读)→资讯(缩略图+作者卡片)→深度/研究(付费)。
- **会员按"研究维度"分层非荐股**：见闻VIP(宏观策略)/见智(个股深度)/**大师课(方法论框架)**/**脱水研报(研报精华摘要)**——付费墙内卖"框架/解读"不卖买卖信号。
- **见闻早报**："5 分钟卡片晨读"+可分享+早餐 FM（养日活+自传播）。
- **宏观日历**；**AI 四维度**：`最新`(秒级)/`头条`(锁定最重要、**过滤噪音只留核心**)/`热文`(穿透阅读权重看市场情绪)/`搜索`(关键词跨时段回溯)。

**可借鉴点（解牛）**
1. **主题追踪=聚焦杀手锏**：关注一个主题→把该主题的 一手快讯+资讯+大师解读 **聚合成一条持续更新的叙事时间线**，命中即推；做成"这件事从头到 now 发生了什么"的还原，不预测走向。
2. **内容分层=合规变现骨架**：免费层=一手快讯(流量/习惯)；差异层=聚焦+大师解读。**"脱水研报"心智直接搬**——大师解读本质=把复杂新闻用某位大师视角**脱水成 3 句**。
3. **AI 四维度当产品框架**：最新=一手 / 头条=A股核心降噪 / 热文=市场情绪 / 搜索=主题回溯；再叠它没有的 **大师视角解读**（同一头条 巴菲特/段永平/索罗斯"会怎么看"，做"多棱镜"非"预测器"）。
4. **每日"解牛早报"**：隔夜/盘前 A股要闻精选+每条一句"大师视角"点睛，卡片一键分享，可选音频。
5. **合规护城河反证**：其关联"选股宝"因非法证券咨询被查、创始人获刑——"不荐股/只展示不预测/只做新闻+解读"是可长期存活的正确赛道。

## 国际：Seeking Alpha · Koyfin · Robinhood — 评级并置 + 记分卡 + 极简走势图

**Seeking Alpha** — "三方评级并置 + 因子记分卡"是产品灵魂
- **三方评级同轴并置**：SA Quant(算法 1–5) / SA Authors / Wall Street 三个判断者叠在**同一条 Strong Sell→Strong Buy 尺**上，一眼读"三方是否一致/分歧"。这个共识/分歧可视化就是 SA 差异化本身。
- **Factor Grades**(A+~F)：Value/Growth/Profitability/Momentum/EPS Revisions；**板块相对**、一键下钻看板块中位数+5 年历史、有淘汰规则、3 年逐日等级迁移热力。
- 文章：文首强制 **Summary Bullets** + 标准化 **Analyst's Disclosure** 块 + 多空框架 + 作者战绩。
- **数据可视化仅 2/5(text-heavy)** —— 正是解牛机会：SA 的分析框架 × Koyfin/Robinhood 的视觉。

**Koyfin** — 干净数据终端，可视化 DNA
- **Percentile Rank 记分卡**：每指标 0–100 分位(估值/利润率/盈利/表现)，双尺(vs 自身 10-20 年历史 & vs 同板块/全球同行)——**纯客观、无买卖含义 = "合规版 Factor Grades"现成范式**。
- `/`命令行+助记符(MYW/MYD)；Watchlist 底部汇总行(均值/分位)+可保存 Views；100+叠加序列+**7 个联动色组**；Market Scatter(右+上=放量涨，板块 ETF→成分股贡献拆解)；暗色=**深藏蓝非纯黑**；主题 6→2；**表格不做单元格渐变**(克制)。

**Robinhood** — 极简移动端，数据压成情绪
- 4 色(电光绿+红+黑+白，红调入橙"让亏损不像报错")；开市白底/闭市黑底(背景即市场状态)。注意美股**绿涨红跌**(与我们反)。
- **招牌单线走势图**：1.5pt 细线**无网格无轴**、随涨跌变色、**末端发光点**、**长按 crosshair 扫读**(顶部大数字实时更新)、**虚线基准线+区域填充**、下方时间药丸 1D/1W/1M/3M/YTD/1Y/5Y/MAX。
- 详情页=**一条长滚动**(渐进披露，不逼看深度)；新闻卡 headline+来源+时间(刻意不放 logo)；分析师评级=买/持/卖百分比条。
- **Snacks**：早 6:30 发 3 条、每条 2-4 句大白话、30 秒读完("像懂行朋友聊天")——把金融讲人话范本。
- **合规反面**：confetti 下单撒彩带被监管点名"游戏化诱导交易"→撤掉。解牛不交易，**恰要避开庆祝/怂恿动效**，动效只做"帮助理解"。色盲开关(Accessible colors)。

**可借鉴点（解牛，按差异化价值排序）**
1.（最高）**「大师共识罗盘」**：把 SA 三方并置搬成 5 persona 并置——**非方向轴**(视角契合度/关注度，绝非买卖轴)；顶部 banner 写"四位大师罕见一致关注"/"分歧点:巴菲特看护城河、林奇看成长拐点"；点圆徽→展开已做的 `InterpretationBody`。解决现在"5-tab 逐个点、看不到全局"。**共识/分歧本身就是内容**，100% 合规。
2.（最高）**「事实记分卡」**：用 Koyfin 分位(客观)非 SA 主观评级。维度只能是事实：估值分位/成长/质地 + **解牛独有资讯维度**(资讯动能=近30天资讯量/重大动态频次、关注热度=followerCount、大师共识度)——关于"新闻"非"价格"的评分，天然合规且竞品给不出。视觉用 **amber 深浅/中性灰阶不用红绿字母**；底部标准化"客观分位非评级非建议"信任块。
3. 解读页排版：文首 **3 条 Summary Bullets(一句话看懂)** + "非投资建议"升级为**标准化披露块**(每篇必现) + 可选多空张力(不下结论)。
4. 走势图按 Robinhood 升级现有 sparkline(**红涨绿跌**)：无网格细线+**末端发光点**+**虚线基准线(昨收)**+区域浅填+**时间药丸**+桌面 crosshair；实体页首屏大数字 hero；**色盲开关**。
5. Koyfin 式**板块贡献可视化**(吃实体图谱红利)：点板块看成分股今日涨跌贡献拆解，纯客观。
6. ⌘K 加助记符+最近浏览；记分卡/罗盘做可复用卡片(延续 RailCard/SectionHead)一致出现。
7. **战略**：解牛独特站位 = SA 观点深度 × Koyfin/Robinhood 视觉克制 × 中国合规(只展示不预测/无点位无买卖)。避开 Robinhood 游戏化动效。
