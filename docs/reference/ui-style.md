# 解牛 — UI 设计语言（2026-07 调研 + 定调）

调研自 2026 年 fintech / 财经资讯 App 设计实践（Bloomberg / Robinhood / 华尔街见闻 / 雪球 / 东方财富 / FT + 多篇 fintech UX 指南）。本文是 UI 重构循环 `jnmvp003` 的北极星。

## 调研结论（2026 fintech 设计共识）

- **中性打底 + 高对比数据 + 精准点缀**：「友好企业蓝」时代结束，转向高对比、粗体排版、克制配色、干净锐利——传达**诚实与透明**。
- **卡片化**：每张卡承载一个指标/数据集，可交互；列表以卡片为基本单元。
- **强视觉层级**：重要数字/标题在顶部用大号高对比字；次要指标在下方用小号灰字。
- **数字排版**：等宽数字（tabular-nums），保证价格/涨跌对齐可读。
- **语义色**：涨跌用色——**中国区红涨绿跌**（与欧美相反），务必按 A 股习惯。
- **导航**：底部 Tab + 可收起头部；点缀色只用在激活态/关键 CTA。
- **状态齐全**：loading 骨架、空状态、错误态是一等公民。
- **质感**：柔和阴影、极细边框、粗体标题；暗色模式是财经 App 标配。

## 解牛的方向：编辑型 · 财经 (Editorial-Fintech)

严肃、可信的财经资讯气质。近黑墨色文字 + 精炼中性灰阶 + 发丝级边框 + 充足留白 + 粗体标题 + 一个精准点缀色（琥珀金，延续品牌、寓意金融、在一片蓝中辨识度高）。红涨绿跌语义。卡片化信息流。

### 设计令牌（globals.css `@theme`）

| token | 值 | 用途 |
|---|---|---|
| `--color-ink` | `#0b0d12` | 标题/正文主墨色（近黑，比纯黑柔和） |
| `--color-brand` | `#f5a623` | 琥珀金点缀（激活态/logo/关键 CTA，**克制使用**） |
| `--color-brand-dark` | `#d98c00` | 点缀 hover/深色 |
| `--color-up` | `#e0322c` | 涨（红） |
| `--color-down` | `#12a36b` | 跌（绿） |
| canvas | `#f7f8fa` | 页面底色 |
| surface | `#ffffff` | 卡片面 |
| 边框 | `gray-200/60` 发丝 | 分隔 |

- 字体：Geist（拉丁）+ 系统中文；正文 `antialiased`；数字 `tabular-nums`。
- 圆角：卡片 `rounded-xl`(12px)、chip `rounded-full`、logo 瓦片 `rounded-[7px]`。
- 阴影：`shadow-sm` 为主，克制。
- 点缀色只占屏幕很小比例——结构靠墨色 + 中性灰。

### Logo（去 emoji）

`public/icon.svg` = 琥珀金**牛头徽标**（横向双角 + 圆盾头 + 负空间口鼻，无卡通眼/嘴）置于近黑圆角瓦片。取「庖丁解牛」之牛（是名词的牛，非牛市）。`Logo` 组件 = 内联徽标 mark + 「解牛」粗体紧排字标。**全 App 不再用 emoji 当 logo/图标**；导航图标用内联描边 SVG（Lucide 风格），persona 视角图标后续改字母 monogram。

## 重构 backlog（每轮一个可验证增量）

1. 品牌识别 + 去 emoji 外壳（logo/mark、TabBar/Bell 描边图标、设计令牌、本文）
2. 首页重构（editorial hero、分区、卡片层级、间距节奏、等宽数字）
3. 资讯卡片系统（来源徽标/时间/tier/实体 chips/hover/行距/密度）
4. 实体页 & 详情页（设计语言、行情卡红涨绿跌、关注区）
5. 发现/我的/通知页一致化（分区标题、空状态、列表）
6. 暗色模式（token + 切换 + 持久化）
7. 打磨（骨架屏、空/错态、过渡、响应式/可访问性、persona 图标去 emoji）
